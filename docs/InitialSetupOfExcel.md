## Supported Formats

This project uses [ExcelDataReader](https://github.com/ExcelDataReader/ExcelDataReader) for reading Excel files.

Supported formats:~~~~

- Binary Excel files (2.0-2003 format; *.xls)
- OpenXml Excel files (2007 format; *.xlsx, *.xlsb)
