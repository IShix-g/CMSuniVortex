## 対応するフォーマット

Excelの読み込み処理に[ExcelDataReader](https://github.com/ExcelDataReader/ExcelDataReader)を使わせていただきました。

対応するフォーマット:
- Binary Excel files (2.0-2003 format; *.xls)
- OpenXml Excel files (2007 format; *.xlsx, *.xlsb)
